# 🚗 Sistema de Análise de Telemetria Veicular

Projeto desenvolvido com foco em análise de dados, pipeline e dashboard interativo.

## 📊 Funcionalidades

* Leitura de dados via CSV
* Cálculo de métricas (velocidade média, RPM)
* Análise de comportamento de condução
* Detecção de aceleração brusca e freadas fortes
* Score de direção
* Dashboard interativo com Streamlit

## 🛠️ Tecnologias

* Python
* Pandas
* Streamlit

## ▶️ Como executar

```bash
pip install pandas streamlit
streamlit run app.py
```

## 📌 Objetivo

Transformar dados brutos de telemetria em informações úteis para análise de desempenho veicular.
